## Master Plan Immutable (designer-locked)

- **Necessary Condition** (NC) — must be true for design to hold
- **Evidence** — observable facts (codebase, industry, prior-art, session-observation; never designer)
- **Rule** — designer-directed restriction
- **Permission** — designer-directed relief from a Rule
- **Risk** — hazard with basis pointer
- **Problem statement** — designer's confirmed hypothesis
- **Phase 4a** / **Phase 4b** — historical names; pivot retires the boundary for cluster C scope but names preserved as paper trail

## Cluster A Shipped (resolution-claim layer)

- **Resolve Condition** (RC) — observable resolution criterion anchored to one Concern; carries five attributes (observable, designer-ratified, problem-statement-anchored, forward-looking, non-restrictive)
- **Concern** — designer-ratified named aspect of the problem statement; RCs anchor here for coverage

## Cluster B.1 Shipped (boundary contract)

- **Caller** — upstream agent submitting material into the proof
- **Submission material** — caller-output shape: `problem_statement` + `elements[]` of typed candidates
- **One-shot submission** — single boundary crossing per proof opening; in-band re-submission forbidden
- **Permissive boundary** — boundary accepts any shape; rigor lives 4b-internal
- **Restructuring** — discrete 4b-internal phase between submission and open
- **Restructuring action label** — closed set: verbatim-preserve / reshape / gap-fill / infer / derive
- **Provenance** — per-element source citation + action label + reasoning chain
- **Open gate** — verifies labels + provenance before transitioning to open

## Cluster B.2 Shipped (closure layer)

- **Closing argument** — single self-contained read derived from current proof state; walks every live RC
- **Two-yes closure gate** — designer must consent to view AND say "go" in same round; mutation clears both flags
- **Phantom element** — withdrawn NC or RC carried forward with closed-set disposition
- **Friction** — tension between two existing elements; four detection shapes; closed disposition set
- **Disposition** — closed-set tag on phantoms (consolidated / superseded / found-redundant / found-incorrect / scope-removed) and on friction (lived-with / relieved-by-exception / dissolved-by-revision / dissolved-by-scope-cut / not-really-friction)
